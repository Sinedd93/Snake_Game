# 🐍 Snake Game (C# WinForms)

A classic Snake game built with C# and Windows Forms.

## 🎮 Features

- Growing snake mechanics
- Random food generation
- Score counter
- Level system
- Obstacles after level 2
- Enemy that chases the player
- Background music

## 🛠 Technologies

- C#
- .NET Framework
- Windows Forms

## 🚀 How to Run

1. Clone the repository
2. Open `Snake #5.sln` in Visual Studio
3. Press Start (F5)
